### 一、前置准备（铺垫阶段）

1. 环境要求

- JDK 安装与配置（需 JDK 8+，验证 java -version）

- 终端 / CMD 基础操作（cd、dir/ls 等）

- XML 基础（标签、属性、嵌套结构识别）

1. 工具准备

- Maven 官网下载（稳定版选择）

- IDE 准备（IDEA/Eclipse，后续集成用）

### 二、核心基础（必学，占日常使用 80%）

#### 1.  Maven 核心认知

- 作用与价值（统一结构、自动化构建、依赖管理）

- 三大核心要素（POM、坐标 GAV、仓库）

- 「约定优于配置」思想（核心设计理念）

#### 2. 环境搭建与配置

- Maven 安装与解压（路径无中文 / 空格）

- 环境变量配置（MAVEN_HOME + PATH 配置）

- 核心配置文件（settings.xml 作用与优先级）

- 本地仓库路径修改

- 镜像配置（阿里云镜像加速）

#### 3. 标准项目结构

- 核心目录划分（src/main、src/test、target 等）

- 目录功能说明（源代码、资源文件、测试代码存放规则）

#### 4. 核心构建生命周期与命令

- 三大生命周期（清洁、默认、站点）

- 默认生命周期核心阶段（compile→test→package→install→deploy）

- 高频命令实操（含组合命令）

#### 5. POM.xml 核心配置

- 基本信息（坐标 GAV、打包类型 packaging）

- 依赖配置

- 依赖范围（compile/test/provided/runtime）

- 属性配置（<properties> 统一版本管理）

### 三、进阶技能（解决复杂场景）

#### 1. 依赖管理深化

- 依赖传递规则（间接依赖自动下载逻辑）

- 依赖冲突场景与解决（排除依赖、版本锁定）

- 依赖查看工具（mvn dependency:tree 命令）

#### 2. 多模块项目管理

- 聚合（标签，多模块一键构建）

- 继承（配置复用）

- 父项目与子项目关系（打包类型 pom 说明）

#### 3. 插件使用

- 核心插件介绍（编译、测试、打包、文档插件）

- 插件配置方式（<build>> 标签）

- 常用插件实操（指定 JDK 版本、跳过测试等）

#### 4. 私服配置与使用

- 私服作用（缓存、私有依赖存储）

- settings.xml 私服配置（镜像、账号密码）

- 依赖上传与下载（mvn deploy 命令）
